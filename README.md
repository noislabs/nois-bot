# NOIS BOT

### Beta test version

This bot insert offchain data Drand to onchain data Nois contract.

### Installation

- Install Node.js >= 15
- mkdir bot folder and `npm install`
- create a `.env` file and follow `.env.example`

### How to use

```
node index.js
```
